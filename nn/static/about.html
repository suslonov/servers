What is this

This site is a collection of several neural network NLP models, pretrained with different sets of parameters.

Models in the collection are designed for simulate human ability to quickly see the text and form some opinion “at glance”. (Or may be it is more similar to dog's ability to understand their master's commands, if dog could receive written commands.)

These models were trained, some for hours, some for days, using an Nvidia card.

The dataset used was nltk.wordnet, very large semantic net of English words.

Neural networks at the site are not training, they do only classification of given examples.

The parameters you can see in list may be suboptimal. It will take some time to found the best ones.

The task

Problem: build neural network could be trained to classify English sentences according to its domains.

For example, if the neural network sees “colibri has nothing to deal with a pine” input, it should answer what it is about birds, trees, biology but not about cars, humans or credit cards.

So, we need our neural network assign probabilities to the list of possible categories, ideally all correct answers should have topmost probabilities. It is standard softmax output layer.

Because input of the model is sequence, our natural choice is recurrent neural network before softmax. You can see several variants of them on this site.

The sentence our neural network should classify may be represented in two ways: as a sequence of chars and as a sequence of words. Both works, but word-wise approach demands more preprocessing not only at training stage and about five times more storage. Moreover, because word-wise approach implies that the models have a dictionary of tens of thousand words, such model will simply work as a bag-of-words regression and do no generalisation. It works, but it is not interesting.

To be clear, these models do not understand natural language, they are only possibly useful block in building of NL-interface.

How to use this site

1) please enter some short sentence which classification may be interest or funny,

2) select up to 3 NN variants from the list of NNs,

It is recommended to select to compare somewhat similar models:

- or same type models with different parameters,

- or different types models with similar parameters.

You may click to model name in the table to see model details and description.

3) press “Classify” and wait a little,

4) see 3 columns with classification possibilities below the NN list.

The header of each result column is a coded description of the NN model responsible for the column.

Numbers in the columns are possibilities of the related category (softmax output).

You may try following sentences for example:

bird

small eagle

rose on the tree was very nice

flying is for birds

is this a bus?

my brother with a motor

do you want to kiss the sky

colibri has nothing to deal with a pine

...

Or you may see what other people ask: Request log


What is this all for

The NN module build like one of these models planned to be used as pre-classification module in AI user adviser.

It may raise up efficiency of analyzing text data, for example detecting of unusual and fraudulent transactions in bank statements.



The NN framework used is TensorFlow.
