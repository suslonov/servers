The problem: privacy threats

Abusing personal data, spreading fake news, flooding with ads, using a market position to promote political agenda, and even annoying users by unwelcome interface upgrades - there are a lot of scandalous points in social networks.
All this creates a strong demand for providing personal Internet space, safe, free from unwelcomed interventions and forced agenda of any kind, but connected, efficient and user friendly.

According to “Moxie” Marlinspike, “People don’t want to run their own servers, and never will. The premise for web1 was that everyone on the internet would be both a publisher and consumer of content as well as a publisher and consumer of infrastructure.”

Moera project solves this problem.

Moera nodes are hosted on Moera servers, which can be private, corporate (belonging to some company), or central (belonging to Moera itself). A user node can be easily transferred from one server to another. In the Moera network, users can have real-world names and nicknames, both protected from misuse.

use case: Social network

A subscription to someone else’s personal blog and shared events keeps a Moera user in touch with friends – as much as the user wants, without annoying ads and without frivolous, unpredictable, and uncontrollable changes by the service owner.

use case: Personal blogging site

Moera users have their own stand-alone blog together with simple and easy subscription to other blogs at any Moera server. Moera servers are linked but independent. Servers have their own domain names and, if a user doesn’t want to be hosted on some server, there is no problem to choose another host server, while still being able to subscrib to a blog, which is hosted on that server, where the user doesn’t want to be hosted. Moera node can keep your stand-alone site side-by-side with your blog and eliminates any entering barrier.

use case: Personal workspace

Moera node can be provided with a wide set of online tools, including task managers, document editors, table processors etc. Moera server online tools will be similar to Google docs, but Moera users know who keeps their docs (end-to end encryption can be provided, hiding users’ content from the server operator) and can be sure that there is a backup system at service nobody can discontinue.

use case: Secured corporate chat server

Do you want to have a Discord-like or Slack-like team collaboration secured inside your company perimeter? You can easily have it.

use case: Virtual cross-server communities

Moera subscription system provides users with a unique feature: various types of groups and communities can be created for the whole Moera network, across all its servers.

use case: Collaboration server

Since users would be provided with online working tools, it is reasonable to expect that these tools would allow collaboration, so a Moera server together with a company private cloud would constitute a very efficient and secure company-internal collaboration system.
